Welcome to my Running Stats Site!

This website helps me track my runs and track my progress. You can easily sort and visualise the running data to gain insights into the runs.

Run Log

View all the runs in a listed form. Runs can be sorted by date, distance, time, or pace in ascending or descending order.

Pace Calculator

Calculate any missing value: distance, time, or pace. Simply input two values, and the calculator finds the third.

Visualise Runs

Generate bar charts or line charts of the runs between any period, grouped by distance or number of runs. Targets can easily be tracked.
